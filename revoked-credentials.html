Trust & Safety

Revoked Credentials

A public list of wallet addresses whose Trucker Credential Token (TCT) has been flagged as no longer trustworthy.

Why This Page Exists

A TCT lives permanently in a carrier's own wallet — once minted, it can't be deleted or edited, even by us. If a carrier is later found to have provided false information, or is confirmed to have committed fraud or theft, their token itself can't be taken back.

What we can do is maintain this separate, public list. If a wallet address appears here, treat any TCT it holds as no longer valid — regardless of what the token itself shows. Always check this list alongside a wallet's TCT status, not instead of it.
